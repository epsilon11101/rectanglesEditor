Ctrl Functions:

  1. Ctrl + e = Edit (w,h) proportions of the selected rectangle
  2. Ctrl + Shift + e = Change (X,Y) position of the selected rectangle
  3. Ctrl + r = Remove selected element
  4. Ctrl + d = Draw new rectangle
  5. CTrl + z = Restore the last rectangle deleted

Instructions:

  1. Load JSON file
  2. Load Image
  3. If you will edit the element o delet it the first thing you need to do is select the element with one click
  4. Once the element was select you can edit with Ctrl functions
  5. You can change the color of the rectangle,default color is RED
  6. Enjoy it
